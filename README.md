# 謝天｜閱讀密室逃脫

這是一款基於網頁的「閱讀理解密室逃脫」遊戲，主題圍繞在陳之藩的經典散文——〈謝天〉。  
透過五關或十關的解謎與選擇題，玩家將深入探索文章的主旨、結構與文學技巧。

## 遊戲特色

- **首頁選單**：可自由選擇難度與關卡數，再開始遊戲
- **兩種難度**
  - 🌱 新手探索者：課文基本理解、段落大意、重點掌握
  - 🔥 高手挑戰者：深度分析、批判思考、論證結構鑑賞
- **兩種關卡數**
  - 🚪 五關密室：約 3 分鐘，輕鬆體驗（密語：謝天・飲水思源）
  - 🏯 十關迷城：約 5 分鐘，完整挑戰（密語：謝天・飲水思源・得之於人）
- **密碼碎片系統**：答對每關即解鎖一個漢字碎片，集齊全部碎片方能逃出密室
- **解析說明**：每題作答後顯示詳細解析，有助課後複習
- **精美視覺**：深夜星空背景、金色琥珀色調、Glassmorphism 卡片、彩帶動畫

## 如何遊玩

1. 在瀏覽器中直接開啟 `index.html`
2. 在首頁選擇難度（新手 / 高手）與關卡數（5 關 / 10 關）
3. 按下「開始逃脫」
4. 依序答題，答對獲得密碼碎片
5. 答錯可選擇「再試一次」或直接進入下一關（不獲得碎片）
6. 完成全部關卡後查看最終密語與成績

## 使用素材

- 網頁結構：HTML + CSS + JavaScript（無依賴外部框架）
- 字型：ZCOOL XiaoWei、Noto Serif TC、Noto Sans TC（Google Fonts）
- 課文來源：陳之藩〈謝天〉
