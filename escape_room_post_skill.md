# Skill 指南：密室逃脫粉專推廣貼文產生器

本檔案是一份 AI 執行指南（Skill File）。當使用者指示要進行「密室逃脫發文」時，AI 必須讀取此指南並自動為其撰寫粉絲專頁發文與首則留言內容。

---

## 觸發條件
當使用者提到「密室逃脫發文」或「為密室逃脫產生粉專貼文」時，啟動本 Skill。

---

## 執行流程

### 步驟一：分析與資料提取
1. **獲取專案資訊**：
   * **課文名稱**：分析當前專案目錄或 `index.html` 中的課文主題（例如：〈謝天〉、〈摩登土產鳳梨酥〉）。
   * **GitHub 專案名稱 (REPO_NAME)**：執行 `git config --get remote.origin.url` 讀取遠端儲存庫網址，提取出專案的英文名稱。若沒有 Git 儲存庫，則根據課文名稱推測可能的 REPO_NAME，並在留言網址中寫入 `https://littleyi22.github.io/REPO_NAME/` 提示使用者手動替換。
2. **對應五大關卡主題**：
   * 檢視網頁中的題庫或根據該課文的國文教學重點，為貼文量身設計「五大關卡（理解維度）」，並配上合適的 Emoji（如：🔍、🧠、🔬、📅、🌍 等），關卡主題需切合課文精神（例如謝天包含「得之於人、飲水思源」等）。
3. **提取學習效益**：
   * 寫出本遊戲能帶給學生的學習益處（例如：訓練閱讀理解、批判思考、邏輯排序等能力）。

---

### 步驟二：套用標準發文格式
AI 必須仿照以下範本格式進行撰文，輸出時將 `<課文名稱>` 等角括號變數替換為實際課文內容：

#### 📝 粉絲專頁貼文範本：
```markdown
#<課文名稱>閱讀密室逃脫
這次我把〈<課文名稱>〉的課文精華，改編成了一場刺激的線上閱讀密室逃脫！裡面總共有五大關卡等著你來挑戰： 

🔍 第一關：[自訂第一關名稱，例如：現象推論]（[簡短關卡說明]） 
🧠 第二關：[自訂第二關名稱，例如：高階辨析]（[簡短關卡說明]） 
🔬 第三關：[自訂第三關名稱，例如：批判思考]（[簡短關卡說明]） 
📅 第四關：[自訂第四關名稱，例如：邏輯排序]（[簡短關卡說明]） 
🌍 第五關：[自訂第五關名稱，例如：跨域遷移]（[簡短關卡說明]）

透過隨機抽題與解謎，不僅能訓練學生的閱讀理解、邏輯思維與批判思考能力，還能更深刻地了解這篇課文背後所象徵的[課文的核心教育意義或人文精神]！

想知道自己能不能順利收集到所有密碼碎片，拿到最終的「專屬完成證書（老師認證）」嗎？快點進留言區第一則的網址挑戰看看吧！👇👇👇

挑戰成功的同學，別忘了在底下留言曬出你的過關證書截圖喔！🏆

#國中國文 #<課文名稱> #閱讀理解 #密室逃脫 #[自訂課文相關 Hashtag]
```

#### 💬 留言區第一則範本（此部分需獨立區隔，方便使用者複製）：
```markdown
💬 留言區第一則（記得把這段發在留言第一樓）
👇 「<課文名稱>」線上閱讀密室逃脫 遊戲連結點這裡 👇 

https://littleyi22.github.io/REPO_NAME/ 

免下載、免安裝，點開網址直接就能在手機或電腦上玩囉！祝大家順利破關！[自訂與課文主題呼應的Emoji，如 🌾✨]
👉需要傳送訊息收藏網址的老師，請留言 #<課文名稱>
👉麻煩留言的老師順手將我發佈的網址按讚👍，以免被淹沒在留言裡面，謝謝大家。
```

---

## 輸出要求
1. **文筆流暢且吸睛**：貼文語氣必須溫暖、活潑，符合教師與家長/學生社群的口吻。
2. **段落清晰**：貼文與留言一樓必須使用虛線或明確的區塊標示，以便使用者直接複製貼上。
